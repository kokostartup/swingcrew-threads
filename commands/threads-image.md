노션 "2026년 Threads 게시글" DB에서 이미지 상태가 "요청"인 글을 찾아 Chrome에서 Gemini 웹을 열어 이미지를 생성하고 노션에 업로드한다.

## 실행 순서

1. 노션 DB(collection://bb743e71-955f-4909-9f44-2fd97cf58101)를 조회하여 이미지 상태가 "요청"인 항목을 찾는다.
2. 요청된 항목이 없으면 "현재 이미지 생성 요청이 없습니다" 안내.
3. 요청된 항목이 있으면 각 항목에 대해:

   a. 해당 페이지를 fetch하여 **페이지 본문(body)**에서 게시글 텍스트를 읽는다.
   b. 이미지 상태를 "생성중"으로 변경한다.
   c. 게시글 텍스트를 분석하여 이미지 프롬프트를 생성한다 (아래 규칙 참고).
   d. Chrome 브라우저에서 Gemini 웹(https://gemini.google.com)을 연다.
   e. 프롬프트 입력창에 이미지 프롬프트를 입력하고 전송한다.
   f. 이미지 생성 완료까지 대기한다 (최대 60초).
   g. 생성된 이미지를 다운로드한다 (다운로드 버튼 클릭).
   h. 다운로드된 이미지를 노션 페이지의 "이미지" 속성에 업로드하거나 페이지 본문에 이미지 블록으로 추가한다.
   i. 이미지 상태를 "완료"로 변경한다.
   j. 실패 시 이미지 상태를 "실패"로 변경하고 "메모"에 에러 기록.

4. 모든 처리 완료 후 결과 요약을 사용자에게 보고한다.

## 이미지 프롬프트 규칙

### 기본 원칙
- **스타일**: 밝고 선명한 골프 관련 일러스트레이션 또는 사진 스타일
- **구도**: 정사각형(1:1) Threads 최적화
- **톤**: 깔끔하고 프로페셔널하면서도 친근한 느낌
- **텍스트 포함 금지**: 이미지 안에 텍스트를 넣지 않는다
- **브랜드 컬러**: 그린 계열 포인트

### 프롬프트 템플릿
```
A clean, bright golf illustration in 1:1 square format for social media.
[게시글 핵심 주제를 영어로 번역한 설명].
Style: modern, minimal, professional golf aesthetic.
Color palette: natural greens, sky blue, white.
No text or typography in the image.
```

## Gemini 웹 사용 방법

1. Chrome에서 https://gemini.google.com 으로 이동
2. 프롬프트 입력창을 찾아 클릭
3. 이미지 프롬프트를 입력 후 Enter 또는 전송 버튼 클릭
4. 이미지가 생성될 때까지 대기 (로딩 표시가 사라질 때까지)
5. 생성된 이미지에서 다운로드 버튼(⬇️) 클릭
6. 다운로드된 파일 경로 확인

## 노션 이미지 업로드

다운로드된 이미지 파일을:
- 노션 페이지 "이미지" files 속성에 external URL로 저장
- 또는 페이지 본문에 이미지 블록으로 삽입

## 주의사항

- Gemini 웹이 이미 로그인된 상태여야 함
- 이미지 생성에 10~30초 소요
- 한 번에 최대 5개 처리
- 실패 시 1회 재시도
