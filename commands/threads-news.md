웹에서 최신 골프 소식을 검색하여 스윙크루 스타일의 Threads 게시글을 생성하고 노션 DB에 저장한다.

## 실행 순서

### 1. 골프 뉴스 수집
웹 검색으로 최신 골프 소식을 수집한다.

인자가 있으면 해당 키워드로 검색, 없으면 아래 주제를 자동 검색:
- PGA Tour / LPGA / KPGA 최신 소식
- 골프 장비 신제품/규제
- 유명 프로 골퍼 관련 뉴스
- 골프 룰 변경
- 아마추어 골퍼에게 관련 있는 트렌드

### 2. 게시글 생성 (3~5개)
수집한 소식 중 아마추어 골퍼에게 흥미로운 것을 골라 게시글 생성.

#### 구조
```
[속보/소식 한 줄 요약]

[상세 내용 2~3문장]

[스윙크루 관점의 코멘트 1~2문장]

[관련 이모지]
```

#### 톤
- 기본 톤을 유지하되 뉴스 전달이므로 약간 객관적
- "~래" "~라고 하네" 같은 전달 어투 가능
- 아마추어 골퍼 시점에서 "이게 우리한테 무슨 의미인지" 코멘트 추가
- 200~500자

### 3. 노션 DB 저장 (사용자 확인 없이 자동 진행)
생성된 모든 게시글을 노션 "2026년 Threads 게시글" DB(collection://bb743e71-955f-4909-9f44-2fd97cf58101)에 저장한다.

각 항목:
- 제목: "[소식 키워드] - 골프소식"
- 페이지 본문(content): 생성된 Threads 글
- 게시 타입: "골프소식"
- 원본 원고: (비워둠 — 원고 기반이 아니므로)
- 게시 상태: "대기"

### 4. 안내
"골프 소식 게시글이 생성되었습니다. 노션에서 확인 후 '게시 승인' 버튼을 클릭해주세요."
