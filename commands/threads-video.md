스윙크루 유튜브 영상에 대한 코멘트와 링크를 포함한 Threads 게시글을 생성한다.

## 실행 순서

### 1. 영상 정보 확인
- 인자로 유튜브 URL이 제공되면 해당 영상의 제목과 내용을 확인한다.
- URL이 없으면 스윙크루 유튜브 채널의 최신 영상을 검색한다.
- 또는 노션 원고 DB에서 해당 영상의 원고를 찾는다.

### 2. 코멘트 게시글 생성

#### 구조
```
[영상 핵심 내용을 훅으로 — 호기심 유발]

[왜 봐야 하는지 1~2문장]

[영상 링크]

🔗 스윙크루
```

#### 톤
- 영상 내용의 핵심 가치를 짧게 전달
- "이번 영상에서는~" 같은 딱딱한 시작 X
- 영상 내용에서 가장 충격적/유용한 포인트를 훅으로
- 150~300자

### 3. 노션 DB 저장 (사용자 확인 없이 자동 진행)
노션 "2026년 Threads 게시글" DB(collection://bb743e71-955f-4909-9f44-2fd97cf58101)에 저장한다.

- 제목: "[영상제목] - 영상코멘트"
- 페이지 본문(content): 코멘트 글 + 영상 링크
- 게시 타입: "영상코멘트"
- 원본 원고: 해당 원고 릴레이션 (있으면)
- 게시 상태: "승인" (영상 업로드 직후 바로 게시 필요)

### 4. 즉시 게시 안내
"영상 코멘트가 '승인' 상태로 저장되었습니다. 다음 자동 게시 시간에 Threads에 올라갑니다. 바로 게시하려면 `/threads-publish`를 실행하세요."

## 예시

```
드라이버 뒤땅 치는 사람 여기 다 모여

오늘 영상에서 딱 하나만 바꿨는데
뒤땅이 완전히 사라진 드릴 공개했어

5분짜리니까 연습장 가기 전에 꼭 봐

👉 https://youtube.com/watch?v=xxxxx

🔗 스윙크루
```

## 스케줄
- 매주 화요일 저녁 8시, 금요일 저녁 8시 영상 업로드
- 영상 코멘트는 업로드 직후 생성 → 즉시 게시 권장
