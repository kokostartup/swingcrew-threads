# Threads 글 스타일 가이드

## 톤 & 보이스

### 기본 톤: "옆자리 싱글 골퍼 형이 알려주는 느낌"
- 반말 사용 (단, 비하/조롱 X)
- 전문 용어는 쉽게 풀어서
- "~했거든" "~인 거 알아?" "~해봐" 같은 구어체
- 이모지 적절히 사용 (과하지 않게, 1~3개)

### 피해야 할 것
- 딱딱한 존댓말 ("~입니다", "~하십시오")
- 과도한 이모지 도배
- 너무 긴 문장
- 전문 용어만 나열
- 광고성 멘트 ("지금 바로!", "놓치지 마세요!")

## 첫 줄(Hook) 공식

### 효과적인 훅 패턴
1. **반전형**: "프로들이 절대 안 알려주는 비밀 하나"
2. **질문형**: "드라이버 200야드도 안 나가는 진짜 이유 알아?"
3. **숫자형**: "3개월 만에 핸디캡 10 줄인 방법"
4. **공감형**: "연습장에서는 잘 치는데 필드만 가면 망하는 사람?"
5. **충격형**: "지금 너 스윙 10번 중 8번은 틀렸어"

### 피해야 할 훅
- "안녕하세요~" 로 시작
- "오늘은 ~에 대해 이야기해볼게요"
- 너무 긴 서론

## 게시글 타입별 가이드

### 타입 A: 숏폼 (짧은 팁/훅)
원고에서 핵심 포인트를 뽑아 200~500자 짧은 글로 작성.

**특징**: 한 가지 팁, 즉각적 가치, 저장/공유 유도
**글자수**: 200~500자
**출처**: 스윙크루 원고

#### 구조 패턴

**꿀팁형** (저장률 높음)
```
아이언 뒤땅 고치는데 5분이면 충분해

대부분 "머리 들지 마" 이런 얘기 하는데
진짜 원인은 체중이동이야

어드레스 때 왼발에 60% 실어놓고
그 상태로 쭉 유지하면서 쳐봐

뒤땅 바로 사라짐 ⛳
```

**스토리형** (공감률 높음)
```
어제 라운드에서 동반자가 드라이버를 바꿨는데
갑자기 250야드를 넘기더라

비결이 뭔지 물어봤더니
"그냥 힘 빼고 쳤어"

진짜야. 힘을 빼야 거리가 나와
스윙 스피드랑 힘주는 건 다른 거거든
```

**리스트형** (공유율 높음)
```
100타 깨려면 이 3가지만 기억해

1️⃣ OB 안 내기 — 드라이버 대신 유틸
2️⃣ 3퍼트 줄이기 — 2m 퍼트 연습
3️⃣ 벙커 탈출 — 모래 먼저 치기

이것만 해도 5타는 줄어 🏌️
```

### 타입 B: 롱폼 (에세이)
원고 내용을 스토리텔링 에세이로 재구성. 하나의 긴 글.

**특징**: 깊이 있는 내용, 스토리텔링, 개인 경험 녹여내기
**글자수**: 1,500~3,000자
**출처**: 스윙크루 원고
**원고 1개당**: 1개 생성

#### 구조
```
[강력한 훅 — 1~2줄]

[공감 유도 — 독자의 고민을 정확히 짚기]

[본론 — 원고 핵심 내용을 스토리로 풀어내기]
- 단락 구분을 명확히
- 중간중간 짧은 문장으로 리듬감
- 구체적 숫자/사례 포함

[결론 — 실천 가능한 메시지로 마무리]
```

#### 톤 특이사항
- 숏폼보다 약간 정중한 반말 (에세이 느낌)
- "~거든요" "~더라고요" 가능
- 소제목 사용 가능 (볼드체는 Threads에서 지원 안 됨, 대신 <> 또는 [] 로 감싸기)

### 타입 C: 스레드 체인 (연속 글타래)
하나의 주제를 여러 개의 연결된 글로 나누어 게시. 첫 글에서 호기심 유발, 답글로 이어가는 형식.

**특징**: 높은 조회수, 클릭 유도, 시리즈 느낌
**글자수**: 각 파트 200~400자, 총 3~5파트
**출처**: 스윙크루 원고
**원고 1개당**: 1세트 (3~5개 연결 글)

#### 구조
```
[파트 1 — 훅 + 문제 제기]  1/n
↓ (답글)
[파트 2 — 핵심 설명/원인]  2/n
↓ (답글)
[파트 3 — 해결책/결론 + CTA]  3/n
```

#### 예시
```
파트 1 (1/3):
골프 시작한 지 1년 됐는데
아직도 100타가 안 깨진다고?

대부분 스윙을 고치려고만 하는데
진짜 문제는 거기가 아니야

내가 100타 깬 순서는 이랬어  1/3

---
파트 2 (2/3):
제일 먼저 한 건 드라이버를 내려놓은 거야

티샷을 유틸로만 쳤더니
OB가 라운드당 3개에서 0개로 줄었거든

그다음은 퍼팅
2m 이내 퍼트만 매일 30분씩 연습함  2/3

---
파트 3 (3/3):
마지막으로 코스 매니지먼트

핀을 직접 노리는 대신
그린 중앙만 노렸더니
더블보기가 확 줄었어

스윙보다 전략이 먼저야 ⛳  3/3
```

#### API 게시 방법
1. 첫 글을 일반 게시 (creation_id 획득)
2. publish 후 post_id 획득
3. 이후 파트들은 `reply_to_id=<post_id>` 파라미터로 답글 게시
```bash
# 파트 2 이후
curl -X POST "https://graph.threads.net/v1.0/me/threads" \
  -H "Authorization: Bearer $TOKEN" \
  -d "media_type=TEXT" \
  -d "text=<파트2 내용>" \
  -d "reply_to_id=<첫글 post_id>"
```

#### 노션 저장 방식
- 제목: "[원고제목] 체인 #파트번호"
- 페이지 본문: 모든 파트를 구분선(---)으로 나누어 저장
- 게시 타입: "체인"
- 게시 시 첫 파트부터 순서대로 자동 게시

### 타입 D: 골프 소식 (뉴스/트렌드)
원고가 아닌, 최신 골프 소식을 전하는 게시글.

**특징**: 시의성, 정보 전달, 채널 권위성 확보
**글자수**: 200~500자
**출처**: 웹 검색 (골프 뉴스)

#### 구조
```
[속보/소식 한 줄 요약]

[상세 내용 2~3문장]

[스윙크루 관점의 코멘트 1~2문장]

[해시태그/이모지]
```

#### 예시
```
PGA 투어에서 드라이버 비거리 제한 규정을 검토 중이래 🏌️

평균 비거리가 300야드를 넘기 시작하면서
코스 설계 자체가 의미 없어지고 있다는 거지

솔직히 아마추어한테는 해당 없는 얘기긴 한데
장비 규제가 시작되면 우리한테도 영향이 올 수 있어

어떻게 생각해? 댓글로 알려줘 👇
```

#### 소식 수집 방법
- 웹 검색으로 최신 골프 뉴스 수집
- PGA Tour, KPGA, LPGA, 장비 소식 등
- 아마추어 골퍼에게 관련 있는 뉴스 우선

### 타입 E: 영상 코멘트 (유튜브 홍보)
매주 화/금 저녁 8시 업로드되는 스윙크루 유튜브 영상을 소개하는 게시글.

**특징**: 영상 트래픽 유도, 짧고 강렬한 티저
**글자수**: 150~300자
**출처**: 유튜브 영상 제목 + 내용
**게시 시점**: 영상 업로드 직후 (화 20:00, 금 20:00)

#### 구조
```
[영상 핵심 내용을 훅으로]

[왜 봐야 하는지 1~2문장]

[영상 링크]

🔗 스윙크루 유튜브
```

#### 예시
```
드라이버 뒤땅 치는 사람 여기 다 모여

오늘 영상에서 딱 하나만 바꿨는데
뒤땅이 완전히 사라진 드릴 공개했어

5분짜리니까 연습장 가기 전에 꼭 봐

👉 https://youtube.com/watch?v=xxxxx

🔗 스윙크루
```

#### 노션 저장 방식
- 제목: "[영상제목] - 영상코멘트"
- 페이지 본문: 코멘트 글 + 영상 링크
- 게시 타입: "영상코멘트"
- 게시 상태: 바로 "승인"으로 설정 (즉시 게시 필요)

## CTA (Call to Action) 패턴

강하지 않은 자연스러운 CTA:
- "다음에 연습장 가면 한번 해봐"
- "이거 저장해두고 필드 가기 전에 봐"
- "궁금한 거 있으면 댓글 남겨"
- "같은 고민 있는 친구한테 공유해줘"

## 게시 타이밍 (참고)

한국 Threads 최적 게시 시간:
- 평일: 밤 9시~11시 (퇴근 후 여유 시간)
- 주말: 오전 8시~10시 (라운드 전) / 오후 7시~9시 (라운드 후)
- 영상 코멘트: 화/금 저녁 8시 (영상 업로드 직후)
